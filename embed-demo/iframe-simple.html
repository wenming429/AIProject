最简单集成 - iframe 方式

复制以下代码到您的网页即可完成集成

HTML 代码：

<!-- 聊天窗口容器 --> <div id="chat-container" class="chat-container"> <iframe src="https://your-im-server.com/#/embed"></iframe> </div> <!-- 悬浮按钮 --> <button class="chat-button" onclick="toggleChat()">💬</button> <!-- 样式 --> <style> .chat-container { position: fixed; bottom: 100px; right: 30px; width: 380px; height: 600px; background: white; border-radius: 12px; box-shadow: 0 8px 32px rgba(0,0,0,0.2); display: none; z-index: 9999; } .chat-container.active { display: block; } .chat-container iframe { width: 100%; height: 100%; border: none; } .chat-button { position: fixed; bottom: 30px; right: 30px; width: 60px; height: 60px; border-radius: 50%; background: linear-gradient(135deg, #BF0008 0%, #D4000A 100%); box-shadow: 0 4px 20px rgba(191, 0, 8, 0.4); cursor: pointer; border: none; font-size: 24px; z-index: 9998; } </style> <!-- 脚本 --> <script> function toggleChat() { document.getElementById('chat-container').classList.toggle('active'); } </script>
