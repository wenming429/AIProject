🚀 LumenIM 快速集成

只需几行代码，即可将即时通讯功能嵌入到您的OA/CRM系统中

⚡ 快速开始 推荐

在您的网页中添加以下代码，即可嵌入聊天窗口：

<!-- 1. 引入 SDK -->
<script src="lumenim-embed-sdk.js"></script>

<script>
// 2. 初始化（一行代码）
const chat = new LumenIMEmbed({
  serverUrl: 'https://im.yourcompany.com',
  theme: 'light-gray'  // 或 'china-red'
});
</script>

📋 集成步骤

  • 下载 SDK
    将 lumenim-embed-sdk.js 文件复制到您的项目中
  • 引入 SDK
    在需要集成聊天的页面中添加 SDK 引用
  • 初始化配置
    配置服务器地址、主题颜色等参数
  • 完成！
    右下角自动显示悬浮按钮，点击即可使用

⚙️ 配置选项

参数 类型 默认值 说明
serverUrl string - LumenIM 服务地址（必填）
theme string 'light-gray' 主题：'light-gray' 或 'china-red'
position object {bottom:30, right:30} 悬浮按钮位置
size object {width:900, height:600} 窗口尺寸
draggable boolean true 是否允许拖拽窗口
resizable boolean true 是否允许调整窗口大小
authToken string null 自动登录Token（可选）
debug boolean false 调试模式

🔌 API 方法

方法 参数 说明
open() - 打开聊天窗口
close() - 关闭聊天窗口
minimize() - 最小化窗口
switchToMessage() - 切换到消息页面
switchToContact() - 切换到通讯录
openConversation(type, id) type: 'user'|'group', id: string 打开指定会话
setTheme(theme) theme: string 切换主题
destroy() - 销毁实例

📡 事件监听

// 监听未读消息数变化
chat.on('unreadCount', (count) => {
  console.log('未读消息:', count);
});

// 监听新消息
chat.on('newMessage', (message) => {
  console.log('新消息:', message);
});

// 监听窗口打开
chat.on('open', () => {
  console.log('聊天窗口已打开');
});

📝 完整示例

<!-- 在 OA 系统页面中集成 -->
<!DOCTYPE html>
<html>
<head>
  <title>OA系统</title>
</head>
<body>
  <!-- OA 系统内容 -->
  <div class="oa-container">
    <h1>欢迎使用OA系统</h1>
    <p>您的日常工作平台</p>
  </div>

  <!-- 引入 LumenIM SDK -->
  <script src="lumenim-embed-sdk.js"></script>
  <script>
    // 初始化聊天组件
    const chat = new LumenIMEmbed({
      serverUrl: 'https://im.company.com',
      theme: 'light-gray',
      authToken: 'user-jwt-token',  // 可选：自动登录
      debug: true
    });

    // 监听事件
    chat.on('ready', () => {
      console.log('LumenIM 已就绪');
    });

    chat.on('unreadCount', (count) => {
      // 更新 OA 系统未读数显示
      document.getElementById('oa-unread-badge').textContent = count;
    });
  </script>
</body>
</html>
