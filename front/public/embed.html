🚀 LumenIM 悬浮聊天窗口

这是一个将即时通讯功能嵌入到网页的解决方案。 通过悬浮球快速访问聊天功能，不影响正常页面内容。

如何使用

1. 登录后，右下角会自动出现悬浮球

2. 点击悬浮球展开聊天窗口

3. 可以拖拽窗口到任意位置

4. 使用最小化或关闭按钮隐藏窗口

API 控制

可以通过 JavaScript 控制悬浮窗口：

// 打开悬浮窗口
window.floatingChat.open()

// 关闭悬浮窗口
window.floatingChat.close()
注意： 只有登录用户才能看到悬浮球和聊天窗口。
